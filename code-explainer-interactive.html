New to JS? The 6 words you need first
object
A bag of named values: { title, url }. The whole app passes these around.
async / await
Network calls take time. await says "pause here until this finishes," without freezing the rest of the program.
Promise
A placeholder for a value that isn't ready yet (a fetch in flight). You await it to get the value.
array methods
.map() transforms each item, .filter() keeps some — both return a new array, leaving the original alone.
Set
A collection that auto-rejects duplicates and answers "have I seen this?" instantly.
throw
Stop now with an error you wrote, instead of crashing later with a cryptic one.
